Char-Estimate



Try to guess the secret word!









What is the purpose of this game?



CharEstimate is a word game created and developed by Joshua Olivier. This game consists of a section where you have to guess the letter (which can be a vowel or a consonant, but it depends on the word). When you press the "Guess the letter" button, it will tell you whether or not you have guessed the letters correctly. Below that,there is another section that you can use if you want to guess the full word. Think of it as a Wordle. In these two sections there are textboxes that you can utilize in order to write the letter or the full word. At the bottom, you can see the underscores, and those underscores tell you how many letters are in the word. One thing that you have to know is that you can modify the word and the underscores. However, the process needs to be edited with the HTML and CSS from the source code, which is on a separate file. I think you should give it a try because it's a very fun/interesting game.
